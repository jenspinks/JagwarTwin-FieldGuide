# Ciphers & Code-Series

This is the corner of the archive where the artifacts stop merely *depicting* things and start pointing at each other.

Three puzzle-families live here. The backward-chained **Flood Artifacts** (named in Mac Option-key glyphs, anchored to the debut *[[Subject to Flooding]]*). The **numeric catalog IDs** that behave like filing addresses. And the **SA-RA-VI, True-Voice, and gate answer-loops** that close on themselves, the catalog quietly eating its own tail.

The rule for all of them is the same: the meaning lives in the *name*, not the picture. A Flood video looks like abstract water and light, but its title is a scrambled word and its code is the *next* artifact's scrambled title. A photograph of a writing session is really named for its own spreadsheet cell. A drawing of an Egyptian priest closes a three-room loop with a funerary epithet. Read these less like pictures and more like a nervous system, passing signals from one object to the next.

One thing worth saying up front, because it is easy to get lost in here: the goal is not to solve the maze. It is to notice that Roy built one whose deepest layer is *self-reference*, a set of rooms whose only real subject is the map that contains them.
