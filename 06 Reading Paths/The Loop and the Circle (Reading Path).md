# The Loop and the Circle (Reading Path)

> **The thread.** The same shape, spun two ways: return that traps you, and return that brings you home.

This is the core cosmology — the single most load-bearing idea in the project, and the structure underneath everything else. The path moves from the governing distinction down through the question and the split it depends on, then into the song where Roy first names the Circle, the shadow's version of that same song, and the album that runs the whole descent. Take this route if you want the engine, not the surface.

## The route

1. [[The Loop and the Circle]] — the governing distinction: two recursions with the same shape and opposite charge. The Loop returns without transformation; the Circle returns with the self remembered.
2. [[Source vs Reflection]] — the question underneath it: does a reflection return you to the source, or trap you in the image? This is the choice that decides which recursion you're in.
3. [[Duality]] — the root structure the whole thing rests on: the arc is never to erase the split, only to order it correctly.
4. [[The Circle]] — the song where Roy first names the Circle aloud, and calls his favorite song that ever came through him; the founding myth at the close of *33*.
5. [[The Circle (Lucius Version)]] — the same song turned in the shadow's hands: the music-box overture that opens the 2026 descent, the Circle spun the other way.
6. [[Lucius Lullaby (album)]] — the full run of that descent: a Loop held almost to the end, and the work of turning it back toward a Circle.
